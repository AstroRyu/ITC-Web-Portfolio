Payrol

This system is an imitation of a payroll system, replicating functions like wage calculation, deductions, and pay stub generation. It is part of our Practical Exposure to Applied Technologies (PETA) program, allowing us to apply theoretical knowledge in real-world scenarios and gain hands-on experience in software development.

Pre-finals

For our Pre-finals PETA, we were tasked with creating three programs using control structures. The first program checks if a number is positive or negative. The second program maps a number to a month (e.g., inputting "4" gives "April"). The third program takes five user-input values, adds them, and calculates their average. These programs helped reinforce my understanding of control structures and user input handling.
